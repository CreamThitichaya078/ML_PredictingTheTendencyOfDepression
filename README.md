# Predicting the Tendency of Depression from Twitter Posts

โปรเจคนี้เป็นการประยุกต์ใช้ NLP และ Machine Learning กับข้อมูลข้อความจาก Twitter

## รายละเอียด
ทำการ preprocessing ข้อความและสกัดฟีเจอร์ จากนั้นฝึกโมเดล MLP Classifier เพื่อทำนายแนวโน้มภาวะซึมเศร้าของผู้ใช้งานจากข้อความใน Twitter

## เทคโนโลยีที่ใช้
- Machine Learning  
- NLP  
- MLP Classifier  

## วัตถุประสงค์
เพื่อแสดงศักยภาพของ AI ในการวิเคราะห์ข้อมูลด้านสุขภาพจิต
